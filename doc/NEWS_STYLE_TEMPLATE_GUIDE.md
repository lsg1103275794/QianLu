# 新闻风格模板使用指南

## 快速开始

新闻风格提取模板 (`news_style_extraction.yaml`) 专门用于分析和提取新闻报道的写作风格，适用于各类新闻体裁的风格迁移。

## 适用的新闻体裁

### 1. 消息（硬新闻）
- **特点**：简短、客观、时效性强
- **结构**：倒金字塔结构
- **导语**：5W1H要素完整
- **示例**：突发事件报道、政府公告、会议新闻

### 2. 通讯（软新闻）
- **特点**：详细、深入、叙事性强
- **结构**：时间顺序或逻辑分类
- **导语**：特写式、场景式
- **示例**：人物通讯、事件通讯、工作通讯

### 3. 特稿
- **特点**：深度、文学性、可读性强
- **结构**：灵活多样
- **导语**：悬念式、描写式
- **示例**：深度报道、调查报道、解释性报道

### 4. 评论
- **特点**：观点鲜明、论证充分
- **结构**：论点-论据-论证
- **导语**：开门见山或引用式
- **示例**：社论、评论员文章、短评

### 5. 深度报道
- **特点**：全面、系统、分析性强
- **结构**：多层次、多角度
- **导语**：综合式、问题式
- **示例**：调查报道、解释性报道、预测性报道

## 新闻风格关键要素

### 1. 客观性（最重要）
```
✅ 正确示例：
"据国家统计局数据显示，2024年GDP增长5.2%。"

❌ 错误示例：
"经济增长非常好，达到了惊人的5.2%。"
```

### 2. 准确性
```
✅ 正确示例：
"会议于12月30日上午9时在北京人民大会堂举行。"

❌ 错误示例：
"会议最近在北京某地举行。"
```

### 3. 时效性
```
✅ 正确示例：
"新华社北京12月30日电"

❌ 错误示例：
"前段时间报道"
```

### 4. 信源标注
```
✅ 正确示例：
"国家统计局局长康义表示..."
"据新华社报道..."
"专家分析认为..."

❌ 错误示例：
"有人说..."
"据悉..."（除非确实无法标注具体信源）
```

### 5. 5W1H要素
```
必须包含：
- Who（谁）：事件主体
- What（什么）：事件内容
- When（何时）：时间
- Where（何地）：地点
- Why（为何）：原因
- How（如何）：方式/过程
```

## 不同媒体类型的风格差异

### 传统主流媒体（如新华社、人民日报）
- **语言**：正式、严谨、书面语
- **结构**：严格的倒金字塔
- **信源**：权威、官方
- **数据**：精确、有来源
- **立场**：完全中立

### 商业媒体（如财新、界面）
- **语言**：专业但不晦涩
- **结构**：灵活多样
- **信源**：多元、交叉验证
- **数据**：丰富、可视化
- **立场**：基本中立，有分析

### 新媒体（如澎湃、梨视频）
- **语言**：通俗易懂
- **结构**：碎片化友好
- **信源**：多样化
- **数据**：简化呈现
- **立场**：适度互动

### 自媒体
- **语言**：口语化、亲切
- **结构**：自由灵活
- **信源**：可能较弱
- **数据**：选择性使用
- **立场**：可能有倾向

## 常见问题

### Q1: 如何处理不同新闻体裁的风格差异？

**A**: 模板会自动识别体裁类型（消息/通讯/特稿/评论等），并提取该体裁的特定规范。例如：
- 消息：强调倒金字塔结构、硬新闻导语
- 通讯：关注叙事性、细节描写
- 特稿：分析文学性、深度挖掘
- 评论：识别论证结构、观点表达

### Q2: 模板如何保证新闻的客观性？

**A**: 模板专门分析：
- 客观性程度（纯客观/适度评论/观点鲜明）
- 立场中立性（完全中立/基本中立/有明显立场）
- 价值判断（避免/适度/明确）
- 多方观点呈现（多方声音/单方声音/主导观点）

### Q3: 如何处理数据和引用？

**A**: 模板详细分析：
- 数据密度、类型、来源、呈现方式
- 引语频率、类型、来源、功能
- 信源标注方式和可信度

### Q4: 能否用于国际新闻风格？

**A**: 可以，但需注意：
- 国际新闻可能有不同的叙事习惯
- 信源标注方式可能不同
- 建议使用目标语言的新闻文本作为源文本

### Q5: 如何提高风格迁移的准确性？

**建议**：
1. 选择风格鲜明的新闻文本（>300字）
2. 确保源文本体裁单一
3. 使用大参数量模型（>30B）
4. 在风格指南中强调核心新闻规范
5. 多次尝试，选择最符合规范的结果

## 最佳实践

### 1. 源文本选择
```
✅ 推荐：
- 主流媒体的标准新闻报道
- 体裁明确、结构完整的文本
- 风格特征鲜明的报道
- 长度适中（300-1000字）

❌ 避免：
- 混合多种体裁的文本
- 过短的快讯（<100字）
- 风格不统一的拼接文本
- 非新闻类文本（如广告、软文）
```

### 2. 模型选择
```
推荐模型：
- GPT-4（理解能力强）
- Claude（新闻规范理解好）
- Deepseek（中文新闻效果好）
- 文心一言（国内新闻风格熟悉）

不推荐：
- 小参数量模型（<7B）
- 纯文学训练的模型
```

### 3. 参数设置
```python
# 风格提取阶段
temperature = 0.3  # 较低，确保准确提取

# 内容生成阶段
temperature = 0.5  # 适中，保持新闻规范同时有一定灵活性
```

### 4. 质量检查
生成后检查：
- [ ] 5W1H要素是否完整
- [ ] 信源标注是否明确
- [ ] 数据是否准确（如果有）
- [ ] 语言是否客观中立
- [ ] 结构是否符合体裁规范
- [ ] 是否有主观评价词汇

## 进阶技巧

### 1. 混合风格
如果需要融合多种新闻风格：
```python
# 提取多个源文本的风格
style1 = await extract_style(xinhua_text)  # 官方媒体
style2 = await extract_style(caixin_text)  # 财经媒体

# 在生成时明确要求融合
prompt = f"""
请融合以下两种新闻风格：
1. {style1}（主要风格，占70%）
2. {style2}（辅助风格，占30%）

报道主题：{new_theme}
"""
```

### 2. 特定媒体模仿
针对特定媒体的风格：
```python
# 收集该媒体的多篇报道
media_texts = [text1, text2, text3]

# 提取共同风格特征
common_style = await extract_common_style(media_texts)

# 使用共同特征生成
```

### 3. 体裁转换
将一种体裁转换为另一种：
```python
# 提取消息体裁的风格
message_style = await extract_style(message_text, template="news_style_extraction")

# 提取通讯体裁的风格
feature_style = await extract_style(feature_text, template="news_style_extraction")

# 将消息内容用通讯风格重写
new_content = await transfer_style(
    source_content=message_content,
    target_style=feature_style
)
```

## 示例对比

### 示例1：官方媒体 vs 商业媒体

**官方媒体风格（新华社）**：
```
新华社北京12月30日电 国务院新闻办公室30日举行新闻发布会，
介绍2024年经济运行情况。国家统计局局长康义表示，2024年中国
经济总体回升向好，全年GDP增长5.2%，超出年初预期目标。
```

**商业媒体风格（财新）**：
```
2024年中国经济交出了一份超预期的答卷。12月30日，国家统计局
公布的数据显示，全年GDP增长5.2%，这一数字不仅超过了年初设定
的5%左右的目标，也超出了多数市场机构的预测。
```

**风格差异**：
- 官方：开门见山，信源前置，纯客观陈述
- 商业：适度评价（"超预期的答卷"），数据后置，增加分析

### 示例2：硬新闻 vs 软新闻

**硬新闻（消息）**：
```
本报讯 记者从市教育局获悉，2025年中考改革方案今日正式发布。
新方案将体育分值从50分提高到100分，并增加艺术类考核。
```

**软新闻（通讯）**：
```
"孩子每天都要跑步锻炼了。"家长李女士说起即将实施的中考改革，
既期待又担心。12月30日，市教育局发布的2025年中考改革方案显示，
体育分值将翻倍，这让不少家长开始重新规划孩子的学习时间。
```

**风格差异**：
- 硬新闻：直接陈述事实，无场景描写
- 软新闻：以人物引语开头，有场景感，关注影响
